‚öôÔ∏è Installation

Prerequisites

  • Node.js (v16.0.0 or higher)
  • npm (v7.0.0 or higher)
  • Google Gemini AI API key

Step-by-Step Installation


# 1. Extract the downloaded zip file

# 2. Navigate to the project directory
cd ai-password-generator

# 3. Install dependencies
npm install

# 4. Create .env file and add your Gemini API key
VITE_GEMINI_API_KEY=your_api_key_here

# 5. Start the development server
npm run dev

# 6. For production build
npm run build

‚ö° Configuration

Environment Variables

Create a .env file in the root directory with the following variables:


VITE_GEMINI_API_KEY=your_api_key_here

Build Configuration

The project uses Vite for building. You can modify the build configuration in vite.config.js.

üìñ Usage Guide

Basic Usage

  1. Open the application in your web browser
  2. Use the slider to set desired password length
  3. Toggle character sets (uppercase, lowercase, numbers, symbols)
  4. Click "Generate Password" to create a new password
  5. Use the copy button to copy the password to clipboard

Advanced Features

  • View real-time password strength analysis
  • Check password against common patterns
  • Assess quantum computing resistance
  • Access comprehensive documentation via the menu
